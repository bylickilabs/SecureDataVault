Thank you for considering contributing to Secure Data Vault.

## Guidelines

1. Fork the repository
2. Create a feature branch
3. Write clean, documented code
4. Submit a Pull Request

## Requirements

- Python 3.10+
- Follow PEP8
- Maintain offline-first design
- No telemetry or network features

## Reporting Issues

Please provide:
- OS
- Version
- Steps to reproduce
- Expected vs actual behavior

Security issues must follow SECURITY.md.
